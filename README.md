# Java Spring Boot 實戰：購物車後端與系統架構設計

本專案展示了如何構建一個具備企業級穩定性的 Java 後端系統，重點在於安全性與異常處理機制。

## 🛠 核心技術亮點
*   **全域異常處理 (GlobalExceptionHandler)**：透過 `@RestControllerAdvice` 統一 API 錯誤回應格式，提升系統健壯性。
*   **JWT 安全驗證**：實作前端與後端的安全通訊機制。
*   **Docker 部署**：包含 `Dockerfile` 配置，支援 Railway 雲端平台自動化部署。

## 📂 專案結構
*   `Springboot-cart-backend`: 後端核心代碼 (Java/Spring Boot)。
*   `Springboot-cart-frontend`: 前端展示介面。
